# Changelog

## [1.0.0] - 2026-03-06

### Added
- Initial release of Zegy Sensor Manager.
- Real-time sensor monitoring dashboard with live WebSocket updates.
- Device registry integration with sensor readings.
- Interactive floor plan editor with drag-and-drop sensor placement.
- Analytics page with sensor distribution by device class and area.
- Device health monitoring with availability status.
- Standalone Docker deployment support.
- Home Assistant add-on with ingress support.
